# Movie Ticket Booking System

Link to Blog : https://ishikaishani.hashnode.dev/movie-ticket-booking-using-dsa

## Overview
The Movie Ticket Booking System is a Java-based application designed to facilitate the booking of movie tickets. This system allows users to view available seats, book tickets, and manage bookings efficiently. It includes features for both administrators and regular users, providing a comprehensive solution for movie ticket management.

## Features
1. **User Authentication:** Users can log in with their credentials or create a new account to access the system.
2. **Seat Booking:** Users can view available seats, select seats for booking, and confirm their bookings.
3. **Seat Chart:** Administrators can view a graphical representation of the theater's seating arrangement.
4. **Statistics:** Administrators can view statistics such as the number of tickets sold, occupancy percentage, and total income.
5. **User Information:** Administrators can view booking details for all users or specific users.
6. **Dynamic Pricing:** Ticket prices are adjusted based on factors such as seat availability, ensuring optimal revenue generation.

## How to Run
1. Clone the repository to your local machine.
2. Ensure you have Java Development Kit (JDK) installed.
3. Open the project in your preferred Java IDE.
4. Compile and run the `MovieTicketBookingSystem.java` file.
5. Follow the on-screen prompts to navigate the application.
